Watchtower — Notes module

A unified notes & todos module. Every entry is a note that can become a todo (a tri-state done = null · 0 · 1). Notes are Global (🌐) or linked to a project, wearing that project's color. Two-pane layout: a filterable list + a markdown editor. Ocean-blue glass, matching the live theme. Click rows, checkboxes, scopes, and toolbar toggles — it's lightly interactive.
